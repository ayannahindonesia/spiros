# Spiros

## Environtment Variables
### General :
```
GO111MODULE
SPIROS_TIMEZONE
SPIROS_PORT

optional:
SPIROS_CORS
SPIROS_URL_REWRITE
```
more about GO111MODULE can be found [here](https://github.com/golang/go/wiki/Modules)

### JWT :
```
SPIROS_JWT_EXPIRES
SPIROS_JWT_SECRET
```
### Database :
```
SPIROS_DB_ADAPTER
SPIROS_DB_HOST
SPIROS_DB_PORT
SPIROS_DB_USERNAME
SPIROS_DB_PASSWORD
SPIROS_DB_LOGMODE(true/false)

optional:
SPIROS_DB_SSL(postgres)
SPIROS_DB_MAXLIFETIME
SPIROS_DB_MAXIDLECONNECTION
SPIROS_DB_MAXOPENCONNECTION
```
